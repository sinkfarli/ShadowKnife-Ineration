## 电机编码器
1. 编码器简介：
   - 将信号 或 数据进行编制、转换为可用以通讯、传输和存储的信号形式的设备。

2. 编码器分类：
   1. 机械：
      - 利用可变电阻器检测旋转位置，该可变电阻器的电阻与旋转角度成比例地变化。    
   2. 光学：
      - 使用光电传感器检测光是否通过安装在电机轴上旋转的码盘的径向上的缝隙的方法。
   3. 磁性：
       - 使用磁传感器来测量由安装在电机轴上的永磁体产生的磁场分布的变化。
   4. 电磁感应：
       - 读取安装在电机轴上的感应线圈（励磁线圈）和固定线圈（检测线圈）之间产生的磁场变化。 

3. 什么是电机旋转编码器？
   - 检测电机旋转的编码器称为旋转编码器。

5. 旋转编码器的作用：
   - 把角位移转换成电信号。
  
6. 什么是电机线性编码器？
   - 检测线性位移的编码器称为线性编码器。

7. 线性编码器作用：
   - 把直线位移转换成电信号。
